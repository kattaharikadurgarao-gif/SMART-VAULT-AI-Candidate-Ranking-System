# SMART VAULT

## AI-Powered Candidate Ranking System

SMART VAULT is an intelligent candidate ranking platform built for the Redrob Data & AI Challenge.

## Problem

Traditional ATS systems rely heavily on keyword matching and often miss highly qualified candidates.

## Solution

SMART VAULT uses semantic understanding, candidate skills, experience, behavioral signals, and recruiter engagement metrics to rank candidates based on true job fit.

## Features

- Semantic Candidate Matching
- Hybrid Ranking Engine
- Explainable AI Recommendations
- Recruiter Signal Analysis
- Scalable Candidate Retrieval

## Technology Stack

- Python
- Sentence Transformers
- BGE Embeddings
- FAISS
- Pandas
- Scikit-Learn
